## Terminal loop (every day)

```bash
mkdir -p ~/python-practice/dayXX_topic
cd ~/python-practice/dayXX_topic
vim script.py
# i → type code → Esc → :wq → Enter
python3 script.py
```

**Stuck in vim?** → `Esc` `Esc` → `:q!` → Enter
